Open Case File Command (File menu, main screen)

Use this command to open an existing case file. The opened case file name will appear in the title bar at the top of the screen. When you choose this option, a dialog box will pop-up if you have made changes in your Symptom List since last saving the case file; this box will allow you to save your current changes before opening the case file. When the case file is opened, all Symptom Lists will be updated to reflect the contents of the file. After you select the file to open, there will be a slight delay as Dr. Hahnemanns Assistant formats its Symptom List from the file data.

You can create new case files with the New Case File command.

Shortcuts

Keys: CTRL+O
