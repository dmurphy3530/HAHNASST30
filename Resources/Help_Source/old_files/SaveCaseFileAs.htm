Save Case File As Command (File menu, main screen)

Use this command to save the active case file under a new name. Dr. Hahnemanns Assistant displays the Save As dialog box so you can name your document. This is also a convenient way to make a copy of an existing case file. The new file name will appear in the title bar at the top of the screen.

To save a case file with its existing name and directory, use the Save Case File command.

Shortcuts

Toolbar:

Keys: CTRL+A
