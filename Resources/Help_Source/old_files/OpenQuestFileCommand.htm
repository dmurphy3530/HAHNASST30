Open Quest. File Command (File menu, main screen and questionnaire)

Use this command to open an existing questionnaire file. The opened questionnaire file name will appear in the title bar at the top of the screen. When you choose this option, a dialog box will pop-up if you have made changes in your questionnaire since last saving the questionnaire file; this box will allow you to save your current changes before opening the questionnaire file. See File Open dialog box for a description of the File Open dialog. When the questionnaire file is opened, all questionnaire answers will be updated to reflect the contents of the file. You can create new questionnaire files with the New Quest. File command.
