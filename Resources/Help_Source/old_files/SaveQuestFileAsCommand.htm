Save Quest. File As Command (File menu, main screen and questionnaire)

Use this command to save the active questionnaire file under a new name. Dr. Hahnemanns Assistant displays the Save As dialog box so you can name your document. This is also a convenient way to make a copy of an existing questionnaire file. The new file name will appear in the title bar at the top of the screen. To save a questionnaire file with its existing name and directory, use the Save Quest. File command.
