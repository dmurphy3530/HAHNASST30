Exit Command (File menu)

Use this command to end your Dr. Hahnemanns Assistant session. Dr. Hahnemanns Assistant prompts you to save any case file or questionnaire file with unsaved changes. Note that a "New" file was automatically opened for you when you started the program. If you do not want to use this file, or if you want to discard your unsaved changes to a file you have opened, select the "No" option. To save your changes, select the "Yes" option. This option will automatically save your changes to the file(s) which appear in the title bar at the top of the screen. If you are saving the changes to a "New" file for the first time, a dialog will pop-up which allows you to select a folder and name for the file.

Shortcuts

Mouse: Double-click the application's Control menu button.

Toolbar:

Keys: ALT+F4
